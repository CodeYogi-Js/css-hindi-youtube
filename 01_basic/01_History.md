# History of CSS

CSS stands for **Cascading Style Sheets**. It is used to style and design web pages (colors, layout, fonts, spacing, responsiveness, etc.).

In the early days of the web, styling was done directly inside HTML using tags like `<font>`, `<center>`, etc. This made websites messy and hard to maintain.

To solve this problem, **CSS was created to separate content (HTML) from design (CSS).**
